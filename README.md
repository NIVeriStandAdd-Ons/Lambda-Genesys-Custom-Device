## Lambda-Genesys-Custom-Device ##

The Lambda-Genesys-Custom-Device repository holds the source code used by NI to build the Lambda Genesys custom device that shipped with NI VeriStand 2014. This custom device wraps the TDK-Lambda Genesys (Serial) driver to allow interaction with a Lambda Genesys DC Power Supply directly from VeriStand.

### LabVIEW Version ###

The code is provided for LabVIEW 2014.

### Built Availability ###

A built version of the Lambda Genesys custom device shipped with NI VeriStand 2014 and earlier. 

### Quality, Limitations ###

This IP has been tested with the versions of NI VeriStand that it was officially released with. Use of this custom device with other versions of VeriStand is not officially supported by National Instruments.

### Dependencies ###

This custom device relies on the TDK-Lambda Genesys (Serial) driver available on the NI Instrument Driver Network.
